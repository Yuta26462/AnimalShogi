# AnimalShogi

## 動作環境
 **dxlib**が`C:\PG\dxlib`にある事。
 
## 操作方法
**マウス右クリック**で選択  
**Qキー**でポーズメニュー

## 設定

* 画像変数  
  TitleImage  
  StageImage  
  Live2DStage  
  GameClearImage  
  Flame  
  Button  
  KomaImage    
  
* 音声変数  
  KomaClick  
  KomaNaru  
  StartClick  
  TitleBGM  
  TitleBGM01  
  GameClearBGM
